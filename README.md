# AI-TAM (AI-driven House Layout Generation and Price Prediction)
Репозиторий проекта **AI-TAM**, объединяющего генерацию планировок квартир с помощью диффузионных моделей и предсказание цен на недвижимость с помощью нейросетей

## Структура проекта

* `plan_prediction/` — модули генерации планировок квартир на базе датасета RPLAN
  * `diffusion_samples/` — сэмплы генерации по эпохам и анимация прогресса обучения
  * `diffusion_checkpoints/` — сохраненные веса лучшей и финальной моделей
  * `train_unet_v1.py` — код обучения диффузионной U-Net модели
  * `generate_dataset_images.py` — скрипт генерации изображений для датасета
* `price_prediction/` — предсказание стоимости недвижимости
  * `07449.ipynb` — ноутбук с подготовкой данных, обучением кастомной схожестной Mixture of Experts модели (`ThreeLayerMoE`) на PyTorch
* `telegram_bot/` — интерфейс Telegram-бота для генерации планировок и инференса модели оценки стоимости недвижимости.
  * `handlers/` — обработчики команд и текстовых сообщений пользователя.
  * `keyboards/` — интерактивные кнопки и меню для удобной навигации.
  * `services/` — модули инференса и бизнес-логики (генерация планов, вызов подмоделей `ThreeLayerMoE`).
  * `utils/` — вспомогательные утилиты для логирования и валидации данных.
  * `config.py` — инициализация настроек проекта и чтение переменных окружения.
  * `main.py` — точка входа для запуска и инициализации Telegram-бота.

## Предсказание стоимости квартир

В проекте используется кастомная модель **ThreeLayerMoE** (Mixture of Experts) на PyTorch для оценки стоимости квадратного метра на основе географических, категориальных и числовых признаков квартир

* Модель использует взвешенные оценки сходства новой квартиры со всей базой данных
* Экспортированные веса и данные для инференса могут быть развернуты в виде веб-приложения на чистом HTML + Javascript без использования бэкенда

## 🤖 Telegram-бот 

Для демонстрации работы разработан асинхронный Telegram-бот, который выступает в качестве интерактивного интерфейса для взаимодействия с нейросетевыми моделями проекта.

Основные функции:

## 1. Apartment Layout Generation

Пользователь может:
- выбрать готовый пример планировки;
- загрузить собственное изображение;
- получить сгенерированный вариант планировки.

## 2. Apartment Price Prediction

Пользователь вводит характеристики квартиры, после чего модель прогнозирует примерную стоимость недвижимости.
